## 1. 单一职责原则

建立两个类 T1、T2，使 T1 完成职责 P1 功能，T2 完成职责 P2 功能。

**相关代码**

```java
public class Main {
    public static void main(String[] args) {
        Cat cat = new Cat();
        Dog dog = new Dog();
        cat.eat();
        dog.run();
    }
}

class Cat {
    public void eat(){
        System.out.println("Cat is eating");
    }
}

class Dog {
    public void run(){
        System.out.println("Dog is running");
    }
}
```
